# copo
A **co**mmand line **po**modoro timer written in Rust.

## Features
 *  Fast.
 *  Simple.
 *  Offline.
 
## Installation
Via cargo;
```
cargo install copo
```

## Usage
```
USAGE:
    copo <work> <rest>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <work>    Duration of work timer in minutes
    <rest>    Duration of rest timer in minutes
```
